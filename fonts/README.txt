DropGain brand fonts
====================

Every .ttf / .otf in this folder loads at startup. No Windows Fonts install needed.

Bundled fonts
-------------
KARNIVBL.ttf          - DROPGAIN wordmark (Karnivore Blue)
Orbitron-Variable.ttf - metric numbers, table headings, nav tabs, metric captions
mionta/mionta.ttf     - optional bundled font (see mionta/)
